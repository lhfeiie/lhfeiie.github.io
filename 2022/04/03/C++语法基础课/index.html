while () 循环语句有多种写法

如：

while (true)

while (cin >> x && x) // 先读入x，再判断x的真假

while (cin >> x, x)// 读入x，’,’后的语句为循环结束的判断条件。

while (scanf(“%d”, &x) != -1)

以上写法都是正确的，其中”scanf (“%d”, &x != -1)” 与”scanf (“%d”, &x)”等价。’‘为取反符号。

在使用条件表达式中，出现’,’时，只有 ‘,’ 后面的表达式起作用，例如：

for (int i = 0; false, i < 10; i++)

该语句中”false”不起作用，而 “i < 10” 会起作用。

在函数内部定义的一个很长的数组时一定要注意，很有可能会报错，因为数组的长度过长会超过栈空间，可以将此数组定义为全局变量，即将其定义在函数外。在函数外定义的数组会将其放在堆空间，堆空间没有任何长度限制，但不能超过内存限制。定义在函数外面的数组变量一定都是0，而定义在函数内部的变量是随机的，此为全局变量（堆）与局部变量（栈）的区别。

stringstream 名称()

stringstream 在#include 中，stringstream（字符串流）可以把一个字符串初始化一个类似于cin（与cin的功能相同）的一个东西，然后从该字符串当中提取出来任意需要的信息（数据类型转换）。

stringstream ssin(s)，将字符串s通过stringstream初始化为一个字符串流，ssin是一个变量，可以随意定义（相当于名称）。

ssin与cin的不同点在于ssin是在字符串当中读入信息。
